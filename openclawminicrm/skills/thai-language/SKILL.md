# OpenClaw Mini CRM — Thai Language Skill

## หลักการ: ภาษาไทยที่เข้าใจง่าย

ทุกข้อความที่ผู้ใช้เห็น **ต้องเป็นภาษาไทย** ที่คนทั่วไปอ่านแล้วเข้าใจทันที
กลุ่มเป้าหมาย = แม่ค้าออนไลน์, เจ้าของร้านค้า, พนักงานขาย — ไม่ใช่โปรแกรมเมอร์

---

## กฏหลัก

### 1. ห้ามใช้ศัพท์เทคนิคโดดๆ
ถ้าจำเป็นต้องใช้ → **ต้องมีคำอธิบายภาษาไทยกำกับ**

| ห้าม | ใช้แทน |
|------|--------|
| Token | คำ (หน่วยที่ AI อ่าน) |
| Sentiment | ความรู้สึกลูกค้า |
| Purchase Intent | โอกาสซื้อ |
| Pipeline | สถานะการขาย |
| Embedding | ค้นหาจากฐานความรู้ |
| Response Time | เวลาตอบกลับ |
| Churn | ลูกค้าที่กำลังจะหาย |
| Lead Score | คะแนนโอกาสซื้อ |
| Dashboard | หน้าหลัก |
| Analytics | วิเคราะห์ |
| Template | ข้อความสำเร็จรูป |
| Broadcast | ส่งข้อความหาลูกค้า |
| Knowledge Base | ฐานความรู้ร้าน |
| Webhook | การเชื่อมต่อ |
| Cron | ทำงานอัตโนมัติ |
| Bot Config | ตั้งค่าบอท |
| Fallback | สำรอง (ใช้ตัวถัดไป) |
| Provider | ผู้ให้บริการ |
| Calls | ครั้งที่เรียกใช้ |
| Loading | กำลังโหลด |

### 3. ข้อความว่าง (Empty State) ต้องแนะนำว่าทำอะไรต่อ

| ห้าม | ใช้แทน |
|------|--------|
| No data | ยังไม่มีข้อมูล |
| No results | ไม่พบข้อมูล |
| Empty | ว่างเปล่า — เริ่มเพิ่มข้อมูลได้เลย |
| Not found | ไม่พบสิ่งที่ค้นหา |

- **LINE** (ไม่ใช่ "ไลน์")

- **Instagram** (ไม่ใช่ "อินสตาแกรม")
- **Telegram**
- **OpenClaw**
- **น้องกุ้ง** (ชื่อ AI — ใช้ภาษาไทยเสมอ)

## วิธีตรวจสอบ

1. เปิดทุกหน้าใน Dashboard
2. มองหาคำภาษาอังกฤษที่ผู้ใช้เห็น
3. ถ้าเป็นศัพท์เทคนิค → แปลเป็นไทย หรือเพิ่มคำอธิบาย
4. ถ้าเป็นแบรนด์ → คงเดิม
5. ถ้าเป็น code/variable → ไม่ต้องแก้
