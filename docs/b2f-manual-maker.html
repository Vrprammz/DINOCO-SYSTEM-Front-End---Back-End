1. ภาพรวม

ระบบ B2F ใช้สำหรับรับใบสั่งซื้อจาก DINOCO ทำทุกอย่างในแอป LINE ไม่ต้องเปิดเว็บแยก — เริ่มจากพิมพ์ @DINOCO ในห้อง LINE Group ของโรงงาน หรือกดปุ่มจากการ์ดแจ้งเตือนที่ระบบส่งมาให้

ขั้นตอนหลัก

  1. DINOCO สร้างใบสั่งซื้อ → ระบบส่งการ์ดแจ้งเตือนมาในห้อง LINE ของโรงงาน
  2. โรงงานกดปุ่ม "ยืนยัน" จากการ์ด + เลือกวันส่ง
  3. โรงงานพิมพ์ @DINOCO → เมนู → "แจ้งจัดส่งสินค้า" (ส่งทั้งหมดหรือบางส่วน)
  4. DINOCO ตรวจรับ + ตรวจคุณภาพ
  5. DINOCO โอนเงินให้โรงงาน → ได้รับการ์ดแจ้ง

สิ่งที่โรงงานทำได้

  • ยืนยัน / ปฏิเสธใบสั่งซื้อ (กดจากการ์ดแจ้งเตือน)
  • แจ้งส่งของ (@DINOCO → เมนู → "แจ้งจัดส่งสินค้า")
  • ดูรายการใบสั่งซื้อทั้งหมด + ประวัติ (@DINOCO → เมนู → "รายการใบสั่งซื้อ")
  • ขอเลื่อนวันส่ง (@DINOCO → เมนู → "ขอเลื่อนกำหนดส่ง")
  • ดูสถานะการจ่ายเงิน (จากหน้ารายละเอียดใบสั่งซื้อ)

DINOCO แก้ไขใบสั่งซื้อรายการที่เปลี่ยน + ปุ่มยืนยันใหม่

11. ถาม-ตอบ

@DINOCO พิมพ์ยังไง?

พิมพ์ @ ในช่องข้อความของห้อง LINE แล้วจะมีรายชื่อขึ้นมา เลือก "DINOCO B2B System" แล้วกดส่ง

ต้องเปิดเว็บไหม?

ไม่ต้อง ทำทุกอย่างในแอป LINE ได้เลย เมื่อกดปุ่มจากการ์ด หน้าเว็บจะเปิดขึ้นมาใน LINE

ส่งของบางส่วนก่อนได้ไหม?

ได้ ส่งกี่รอบก็ได้จนกว่าจะครบ

ถ้า DINOCO บอกว่าไม่ผ่าน QC ต้องทำอะไร?

รอ DINOCO เลือกวิธีจัดการ — ถ้าเลือก "ส่งทดแทน" จะเห็นสถานะกลับเป็น "รอผลิต" แล้วพิมพ์ @DINOCO → "แจ้งจัดส่งสินค้า" เพื่อส่งของอีกรอบ

DINOCO จ่ายเงินเมื่อไหร่?

หลังตรวจรับครบ ตามเงื่อนไขเครดิต (เช่น 30 วัน) จะมีการ์ดแจ้งเมื่อโอนแล้ว

ดูยอดค้างจ่ายได้ที่ไหน?

พิมพ์ @DINOCO → "รายการใบสั่งซื้อ" → กดที่รายการ → ดูส่วน "เครดิตโรงงาน"

ขอเลื่อนวันส่งได้กี่ครั้ง?

ไม่จำกัด แต่ DINOCO ต้องอนุมัติทุกครั้ง

ส่งเกินจำนวนที่สั่งได้ไหม?

ไม่ได้ ระบบจำกัดจำนวนส่งไม่เกินค้างส่ง

ทำไมได้รับการ์ด "ยกเลิก"?

DINOCO ยกเลิกใบสั่งซื้อ จะแจ้งเหตุผลมาด้วย ใบนี้ไม่ต้องดำเนินการต่อ
